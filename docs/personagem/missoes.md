# Missões

Cada andar do jogo possui uma **missão única** com objetivos e recompensas próprias. As missões são a principal forma de obter equipamentos especiais.

---

## Como Funciona

1. Acesse a tela de missões no andar em que está.
2. A missão exibe os **itens necessários** e a quantidade de cada um.
3. Se você possui todos os itens, o botão de realizar missão fica disponível.
4. Ao completar, os itens são **consumidos** e você recebe as recompensas.

---

## Requisitos

- Você precisa estar **livre** — não é possível completar missões durante uma aventura.
- Todos os itens exigidos devem estar no seu inventário.

---

## Recompensas

Cada missão concede:

- **Experiência (XP)**
- **Moedas**
- **Itens** — geralmente equipamentos exclusivos daquela missão

Se houver bônus de XP ou moedas ativos no servidor, eles são aplicados às recompensas.

---

## Repetição

Se você possui itens suficientes, pode completar a mesma missão **várias vezes de uma só vez**, escolhendo a quantidade desejada. Não há limite diário.
