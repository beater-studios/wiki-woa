# 26-12-2023 | Versão 1.0

**Introdução**

* As atualizações agora serão nomeadas por versão, iniciando com a 1.0.0.

**Mudanças para Novos Jogadores (tks: @elementclient):**

* **Lobo Frenético**: ajustes nos atributos:
  * Força: de 8 para 7;
  * Inteligência: de 5 para 2;
  * Vigor: de 5 para 3.
* sistema de desarme: reduzida a chance de desarmar jogador baseada na vida do monstro;
* novas contas iniciam com 2.000 moedas (antes 500).

**Mudanças Gerais**

* removida a restrição de nível mínimo para atacar monstros (tks: @chavescast);
* alterações no Ranking:
  * exibição limitada aos TOP 50 jogadores;
  * critérios de ordenação: experiência, vitórias, derrotas (menos é melhor), e nome (alfabeticamente).
* lista de ações para batalha: itens de regeneração agora organizados pelo poder de cura e quantidade;
* ao ganhar batalhas contra monstros:
  * exibição da quantidade de itens que jogador possui ao lado do item dropado.
  * removido texto “Você ganhou” ao lado do XP ganho.
* restrições durante Aventuras (tks: @jpsimplicio):
  * uso de itens de regeneração pelo inventário;
  * compra de equipamentos e itens de regeneração.
* ordenação de inimigos na visualização da lista para batalhar agora baseada no nível do monstro (mantendo Chefes de Andar no final).

**Correções**

* corrigido o ataque com "Soco" que não estava acertando golpe (tks: @reiwise);
* acesso a regiões sem quest agora exibe texto descritivo (tks: @lelel\_);
* correções de ortografia.

Outras mudanças também aplicadas para optimização e correção de bugs no jogo.
