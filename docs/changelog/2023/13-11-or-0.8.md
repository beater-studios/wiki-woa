# 13-11-2023 | Versão 0.8

**Novidades:**

* descrição nova para item "Katana" (por: @.wesley\_\_);
* alterações no nome das telas no menu de batalha (tks: @chavescast):
  * "Caçar" agora se chama "Aventura" e todos textos que faziam referência a "caça" foram trocados para o contexto de "aventura";
  * "Boss" agora se chama "Caçada" (tks: @_bruno\_costa_).

**Alterações:**

* chefes de andar aparecem por último na lista horizontal de mobs para atacar;
* alterações sobre precisão:
  * todos itens que não são armamentos foram definidos precisão para 0;
  * todos armamentos que estavam maior que 100% foram definidos para 100% pois é o máximo;
  * no inventário ou loja a informação de precisão só aparece para itens que são armamentos.

**Correções:**

* cálculo de crítico estava considerando número errado surtindo em causar crítico mais vezes do que o normal;
* correções de ortografia nos textos das aventuras.

Outras mudanças e correções também aplicadas para optimização e correção de bugs.
