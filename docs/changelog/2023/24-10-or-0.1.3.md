# 24-10-2023 | Versão 0.1.3

**Mudanças:**

* _**Bracelete Luminoso**_ aumentou preço para **8.000** moedas e _**Bandana de Maid**_ caiu preço para **1.000**;
* desabilitado habilidade "_**Pesca**_" temporariamente no jogo devido que está sem uso, e todos pontos de habilidades utilizados nela pelos jogadores serão convertidos de volta para uso com _**bônus de 2 pontos**_ como desculpas pelo inconveniente.

**Correções:**

* quando jogador morria para Chefe de Andar enquanto usava item de regeneração:
  * não upava nível se atingia XP suficiente;
  * resetava quantidade de vitórias contra o chefe.

Outras correções e optimizações também foram aplicadas.
