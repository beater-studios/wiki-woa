# 27/04 | 1.5.10

Nesta atualização, otimizamos massivamente as imagens do jogo e adicionamos uma nova quest preparatória para o confronto com o Illfang.

**Novidades:**

* **Nova quest: "Preparação para a Tempestade"** — disponível no 1º Andar. Reúna Armaduras Kobold para ajudar na preparação do grupo que enfrentará o Illfang. Recompensa: 12.000 XP e 10.000 moedas;
* Item "Armadura Kobold" renomeado e com descrição atualizada.

**Mudanças:**

* **Imagens convertidas para formato AVIF** — mais de 1.700 imagens de fundo foram convertidas de WebP para AVIF, reduzindo significativamente o tamanho dos arquivos e melhorando o tempo de carregamento do jogo;
* Texto "Quest Finalizada" alterado para "Missão Finalizada" por consistência;
* Mensagem de itens insuficientes para completar quest melhorada: agora exibe "Você não possui todos os itens necessários!" em vez do texto em caixa alta anterior;
* Corrigido o cálculo de conclusão múltipla de quests para considerar corretamente o item com menor quantidade disponível.
