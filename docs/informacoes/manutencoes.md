# Manutenções Programadas

Periodicamente o jogo entra em modo de manutenção para aplicar atualizações, correções e melhorias. Durante esse período, o acesso ao jogo fica temporariamente suspenso.

---

## O Que Acontece

- Todos os jogadores são redirecionados para uma página de manutenção.
- Ao término, o acesso é restaurado automaticamente e você retoma de onde parou.

---

## Como Ficar Informado

As manutenções são anunciadas com antecedência no servidor oficial do [Discord](https://discord.swordartonline.com.br).
