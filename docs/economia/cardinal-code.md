# Cardinal Code

Cardinal Codes são códigos promocionais que concedem recompensas ao serem resgatados. Os códigos são distribuídos em eventos, promoções e ações especiais.

---

## Como Resgatar

1. Acesse **Configurações** → **Resgatar Cardinal Code**.
2. Digite o código no campo indicado.
3. Clique em **Resgatar Código**.
4. As recompensas são aplicadas imediatamente ao seu personagem.

---

## Recompensas

Cada código pode conceder uma ou mais recompensas:

- **Experiência (XP)**
- **Moedas**
- **wCoin**
- **Itens**
- **Acesso a andares**

---

## Regras

- Cada código pode ser usado **uma vez por jogador**.
- Alguns códigos possuem **limite de usos** — quando atingido, o código não pode mais ser resgatado.
- Códigos podem ter **data de validade** — após a expiração, não é mais possível resgatá-los.

---

## Histórico

A tela de resgate exibe os **últimos 3 códigos** que você resgatou, com a data e as recompensas recebidas.
